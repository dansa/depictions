This is the official beta for the iOS9 remake of Flite
At the moment it's extremely incomplete, but I wanted to get it up and running as soon as possible so that people can keep track of progress and enjoy the theme while I'm working on it!

I have only tested the theme in Anemone, but I believe it should work fine in Winterboard as long as you have IconBundles installed
There is no @3x support yet, so if you're running a device with higher pixel ratio icons might look a bit.. Bad? @3x support will be added in the future but right now I'm focusing on getting icons done!

If you have any questions or feedback, I reccomend contacting me on Twitter @dansaDisco!
